Why Kimi mentions Grok:
- Grok/XAI is one common provider in Kimi’s login menu.
- The prompt appears when no provider/model session is configured (e.g., "Model: not set, send /login").

Decision rule:
- If you want to avoid API billing/key entry, always choose the login option that says **Browser** or **OAuth**.
- Only use **API key** login if you explicitly want token/billing via keys.
