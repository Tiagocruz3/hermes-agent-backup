# Product Hunt Launch Checklist

## 4 Weeks Before

- [ ] Create PH account (if new)
- [ ] Join discussions, build karma (comment on 10+ products)
- [ ] Research optimal launch day (Tuesday-Thursday, avoid major events)
- [ ] Prepare 5 gallery images (1270x760px each)
  - [ ] Image 1: Hero dashboard screenshot
  - [ ] Image 2: Feature showcase (content generation)
  - [ ] Image 3: Feature showcase (leads/research)
  - [ ] Image 4: White-label reports
  - [ ] Image 5: Team collaboration view
- [ ] Write maker comment (personal story, 3-4 paragraphs)
- [ ] Record 60-sec demo video (optional but recommended)
- [ ] Prepare first 5 comments (common questions + answers)

## 2 Weeks Before

- [ ] Line up a hunter (or hunt yourself if karma >100)
- [ ] Schedule launch date
- [ ] Prepare supporter outreach list (50+ people)
- [ ] Write supporter DM/email template
- [ ] Soft launch to beta list, collect 3+ testimonials
- [ ] Ensure landing page has PH badge/upvote widget

## 1 Week Before

- [ ] Test all product flows (signup → onboarding → core feature)
- [ ] Prepare launch day social posts (X, LinkedIn, Slack communities)
- [ ] Set up analytics tracking (UTM params for PH traffic)
- [ ] Prepare "thank you" message for supporters
- [ ] Schedule email to list for launch day morning

## Launch Day (Hour-by-Hour)

**6:00 AM PT — GO LIVE**
- [ ] Post goes live
- [ ] Post maker comment immediately
- [ ] Share in 3 communities (Slack, Discord, IndieHackers)
- [ ] Send email to beta list

**7:00-10:00 AM — ENGAGEMENT WINDOW**
- [ ] Respond to EVERY comment within 5 minutes
- [ ] Pin a comment with quick-start guide
- [ ] Share on X/Twitter with screenshot
- [ ] Share on LinkedIn (personal + company)

**10:00 AM — MILESTONE CHECK #1**
- Target: 100 upvotes
- If below: push harder in communities, DM more supporters
- If above: celebrate publicly, momentum attracts momentum

**12:00 PM — SECOND PUSH**
- [ ] Post in 3 more communities
- [ ] Email any segments not yet reached
- [ ] X thread about the launch story
- [ ] LinkedIn post with behind-the-scenes

**2:00 PM — THIRD PUSH**
- [ ] Slack communities (SaaS, marketing, agency groups)
- [ ] Discord servers (buildinpublic, indie hackers)
- [ ] Reddit (r/SaaS, r/marketing, r/entrepreneur) — be careful not to spam

**4:00 PM — MILESTONE CHECK #2**
- Target: 300 upvotes
- If on track for Top 5: keep pushing
- If not: focus on comment quality over volume

**6:00 PM — EVENING**
- [ ] Respond to all remaining comments
- [ ] Thank top supporters publicly
- [ ] Prepare "day after" follow-up post

**11:59 PM — CLOSE**
- [ ] Final comment responses
- [ ] Screenshot final ranking
- [ ] Celebrate, rest

## Day After

- [ ] Thank everyone who supported
- [ ] Analyze traffic (GA/Plausible)
- [ ] Follow up with top commenters personally
- [ ] Write launch retrospective (for blog/LinkedIn)
- [ ] Add PH badge to landing page with ranking

## Post-Launch Week

- [ ] Convert PH traffic: email drip for signups who didn't activate
- [ ] Reach out to press who covered similar products
- [ ] Apply for PH newsletter feature (if Top 5)
- [ ] Update all social bios with "Featured on Product Hunt"
